Code into get_data.py file is tested using python console for checking data and it's validation.
ChroPath Google Chrome extension used to find unique path to element into html documents.
Use of Selenium is stopped, cause it needs to scroll pages up-down, left-right, etc.
Pages are static so html code will not be modified at time of scraping.

User-Agent is used to retrive data. Respective site (adviserinfo.sec.gov) does not allow bots, scripts to collect data.
